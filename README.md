# 1 关于

该仓库是笔者星标过的仓库合集，虽然Github已经自带分类功能，但是仅仅只有仓库列表，也不支持图片以及外链，因此决定专门开一个仓库记录星标过的仓库，同时进行基本的分类，以及添加了方便搜索的标签。
